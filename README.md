# Pandocを使ってMarkdownからPDFを作成するやつ

それなりにスタイルを定義したよ

## 必要な環境

- Pandoc
- Pandoc Crossref
    - texの参照機能を使ってPDFを生成するためのもの
- LaTeX環境
    - 特に、lualatexコマンドが使えるようにしておく
